有一个长度为 $n$ 的数组 ${a_i}$ 。



你可以进行最多 $k$ 次如下操作（也可以不进行操作），每次你选择一个位置 $i(1le ile n)$ ，选择如下两个操作之一，进行操作：





- 加1：令 $a_i$ 变成 $a_i +1$；

- 减1：令 $a_i$ 变成 $a_i-1$。



你希望在操作之后让数组中所有元素的最大值和最小值的差尽可能小，请求出，这个差最小可以是多少？



一次输入只有一组数据，包含两行，第一行包含两个用空格隔开的整数 $n, k(1le nle 2 imes 10^5, 1le kle 10^{18})$。



第二行包含 $n$ 个用空格隔开的整数 $a_i(1le a_ile 10^9)$ 表示数组。为了方便，数组已经从小到大排序，即保证对于所有 $1<i<n$ 满足 $a_i le a_{i+1}$ 。



输出一行一个整数，表示操作之后，数组最大值和最小值的差的最小值。

输入1

```

5 4

1 1 1 2 3

```

输出1

```

0

```

输入2

```

7 6

1 1 1 4 7 8 9

```

输出2

```

5

```



对于样例 1，可以对第 4 个元素进行一次减1，对第 5 个元素进行2次减 1，得到的数组是 ${3,3,3,3,3}$，最大值和最小值的差是 0。



对于样例 2，可以对第 5 个元素进行一次减1，对第 6 个元素进行2次减1，对第 7 个元素进行3次减1，得到的数组是 ${1,1,1,4,6,6,6}$ 最大值和最小值的差是 5。
